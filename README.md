# How to use a Mac textfield

Learn how to use a keyboard shortcuts on textfields on a Mac. You will be guided though a series of tasks involving usage of your keyboard to move cursors, select, or modify text. It is your task to work out whch keyboard shortcuts will achieve this in the most efficient way. There are no correct answers. This is just a way for you to practice.
